
- Bedeutet Ausgeschrieben Procedural Language / PostgreSQL Structured Query Language.

- PL/PG SQL ist die Prozedurale Programmiersprache von PostgreSQL

- PL/PG SQL wird verwendet um Queries mit Variablen, Kontrollstrukturen und Schleifen zu erweitern

- Es erlaubt das Ausführen komplexer Logik direkt auf dem Datenbankserver, was Netzwerk-Transfers zwischen Anwendung und Datenbank minimiert.

- ###### Fazit: Die Programmiersprache, die SQL um echte Logik wie Schleifen und Bedingungen erweitert.
